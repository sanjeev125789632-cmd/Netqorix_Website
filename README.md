# Netqorix — Digital Agency Website

Netqorix is a premier digital agency that turns ideas into powerful software. This repository contains the source code for the Netqorix marketing website — a modern, high-performance single-page site built to showcase the agency's services and capabilities.

## About

Netqorix specializes in building high-performance web applications, elegant UI/UX designs, and scalable cloud solutions. The website presents the agency's brand, services, and value proposition through a fast, responsive, and visually polished experience.

## Features

- Responsive, mobile-first single-page design
- - Modern design system with a custom color palette and Inter Tight typography
  - - Interactive UI details, including a custom animated cursor
    - - SEO-friendly markup with descriptive meta tags
      - - Lightweight, dependency-free build using plain HTML and CSS
